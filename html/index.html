CppLox Language

Name
CppLox
Description
A hobby dynamic typing interpreted language based on the Lox language with my own additions. Currently, has a bytecode-based virtual machine with the typical switch-case dispatch method. I plan to add LLVM Jit Support after the bytecode implementation is complete.
